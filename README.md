# Проект: Mesto. Сделан в рамках обучения на Яндекс.Практикум

***Обзор***

**Описание:**
Проект: Mesto - это интерактивная страница, куда можно добавлять фотографии, удалять их и ставить лайки.

**Функционал:**
* Figma
* Картинки
* Flexbox
* Grid-сетка
* Реакция на действия пользователя
* Медиа-запросы
* Адаптивная верстка

**Технологии:**
Методология БЭМ
HTNL
JS
CSS:
    Flexbox
    Grid Layout
    Positioning
GitHub Pages
Figma
